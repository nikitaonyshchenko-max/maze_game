# Інструкції з запуску гри

Щоб запустити гру, виконайте такі кроки:
1. Клонуйте репозиторій на локальний комп'ютер:
   ```bash
   git clone https://github.com/nikitaonyshchenko-max/maze_game.git
   ```
2. Перейдіть до папки проекту:
   ```bash
   cd maze_game
   ```
3. Встановіть необхідні залежності:
   ```bash
   npm install
   ```
4. Запустіть гру:
   ```bash
   npm start
   ```

# Алгоритми, що використовуються

У цій грі використовуються такі алгоритми:
- **Алгоритм DFS (глибина перший)**: виконує пошук по всіх можливих шляхах до тих пір, поки не знайде ціль.
- **Алгоритм BFS (ширина перший)**: знаходить найкоротший шлях до цілі, досліджуючи всі можливі маршрути.

# Управління грою

- Для переміщення використовуйте стрілки на клавіатурі.
- Для паузи гри натисніть `P`.
- Для перезапуску гри натисніть `R`.
